# Treasure Hunts

Printable treasure hunts and games for kids.

**[Play online](https://berteaux.github.io/games/](https://berteaux.github.io/treasure-hunts-printable/))**

## Available Games

- **Pokopia Treasure Hunt** (FR) -- A 10-step Pokemon-themed Easter treasure hunt for kids. Print, cut, hide, play!

## Usage

1. Open the game in your browser
2. Print (Ctrl+P / Cmd+P)
3. Cut out the cards and hide them
4. Let the kids play!
