Antisèche parents — ne pas imprimer avec les cartes enfants

🔐 Antisèche — Réponses & Instructions

ÉtapeType · Indice pour les enfants bloquésRéponse → lieu suivant
1 · Boîte aux lettres Énigme poétique. Si bloqués : "Je fais cuire les repas, j'ai des feux et des casseroles." CUISINE
2 · Cuisine Code A=1. Si bloqués : donne-leur S=19 pour débloquer le 1er mot. 19·1·12·12·5 = SALLE / 1·13·1·14·7·5·18 = AMANGER → SALLE À MANGER
3 · Salle à manger Message miroir — ils doivent trouver seuls qu'il faut un miroir. Si bloqués après 5 min : "Il y a un truc spécial avec ce texte…" Texte inversé = "Cherchez le meuble de Mimi dans la maison" → MEUBLE DE MIMI
4 · Meuble de Mimi Mots mystère. Le mot caché BAIN est le lieu suivant. Chercher PICHU (vertical col.0), OEUF (horizontal ligne 0), ECLOS (diagonal ↘ depuis col 1). Mot caché = BAIN (horizontal ligne 5) → SALLE DE BAIN
5 · Salle de bain Code symboles Pokémon. Si bloqués : "Chaque symbole = une lettre, utilise la clé." ★♦●●✦♦☾ = CELLIER
6 · Cellier Énigme poétique Bouldeneu. Si bloqués : "Je suis dehors, je suis fleuri." MASSIF
7 · Massif Sudoku 4×4 avec 🔥💧🌿⚡. Les cases vides à compléter. Le mot caché OLIVIER est dans la grille une fois résolue. Ligne 1: 🌿⚡🔥💧 / Ligne 2: 🔥💧⚡🌿 / Ligne 3: ⚡🌿💧🔥 / Ligne 4: 💧🔥🌿⚡ → mot caché = OLIVIER
8 · Olivier Mots mystère. Le mot caché TERRASSE est le lieu suivant. Chercher PIKACHU (horizontal L0), LUGIA (vertical col 4), EVOLI (vertical col 7). Mot caché = TERRASSE (horizontal L5) → TERRASSE COUVERTE
9 · Terrasse couverte QR code → trailer Johto (youtube.com/watch?v=HK6Wsm_9x3I). À 0:23 on voit des légumes / un jardin. Si bloqués après la vidéo : "Regarde dehors." POTAGER
10 · Potager Rébus final. Si bloqués : "Chaque image donne un son, enlève des lettres." 🍅(TOMATE)-MATE + 🐒(SINGE)-SIN + 🏴‍☠️(PIRATE)-RATE = TO + GE + PI = TOGEPI (mais ne pas révéler — le trésor est là !)
Avant de cacher les cartes : générer un vrai QR code sur qr-code-generator.com avec l'URL https://www.youtube.com/watch?v=HK6Wsm_9x3I&t=23 et coller l'image sur la carte 9 à la place du QR code décoratif.

Ordre de cachette : Boîte aux lettres → Cuisine → Salle à manger → Meuble de Mimi → Salle de bain → Cellier → Massif → Olivier → Terrasse couverte → Potager (trésor ici).

Pokémon mystère : Placer la peluche/figurine avec la carte de révélation finale dans le potager. Ne pas mentionner le nom sur aucune carte.
